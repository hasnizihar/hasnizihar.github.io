# Unicode to Bamini Converter

Unicode to Bamini Converter is a professional, modern, and high-performance Tamil Unicode to Bamini font converter. It features a sleek, glassmorphism-inspired interface designed for optimal user experience across all devices.

![Unicode to Bamini Converter Screenshot](https://hasnizihar.github.io/u2b/lib/image.png)

## Features

- **Instant Conversion**: Converts Tamil Unicode text to Bamini font encoding in real-time.
- **Modern UI**: meaningful animations, glassmorphism effects, and a responsive design.
- **Privacy Focused**: All conversion happens client-side; no data leaves your browser.
- **Productivity Tools**: One-click copy buttons and auto-paste functionality.
- **Responsive**: Fully optimized for desktops, tablets, and mobile devices.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Layouts (Grid/Flexbox), Variables, Animations, and Visual Effects.
- **JavaScript (ES6+)**: specialized conversion logic and DOM manipulation.

## Installation & Usage

1. **Clone or Download** the repository.
   ```bash
   git clone https://github.com/hasnizihar/u2b.git
   ```
2. **Open** the project folder.
3. **Launch** `index.html` in any modern web browser.
