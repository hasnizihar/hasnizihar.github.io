## How to Use

1. **Enter URL/Text**: Type the content you want to encode. Use the **Clear** (X) button to reset the field quickly.
2. **Customize**: 
   - Use the **Color** picker to change the QR code color.
   - Use the **Background** picker to change the background color.
   - Adjust the **Size** (in pixels) to fit your needs (clamped between 100 and 1000).
3. **Generate**: The QR code generates live. Click "Generate QR" to force a refresh if needed.
4. **Action**:
   - Click **Download** to save the PNG file.
   - Click **Copy Image** to copy the QR code to your clipboard for quick sharing.
